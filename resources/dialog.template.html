Setting the Serial Port

  1. Make sure that your Diagnostic Cable is plugged into the PC.
  2. Select your Serial Port from the dropdown in Settings and click

MacOS / Linux

Mac and Linux use ports that follow the "/dev" convention. Your Diagnostic Cable will create a new /dev/ port typically of the format /dev/cu.usbserial-FTXXXX or similar.
There may be 2 ports with a similar name listed, a /dev/tty.usbserial... and /dev/cu.usbserial...
Conventionally the /dev/cu. should be used with MemsFCR

Windows

Windows will create a new COM port that is mapped to your Diagnostic cable.
Often this is COM3 or COM5, although you may need to read the Installation Guide for more information if this isn't the case on your PC.

Windows requires the installation of drivers for the Diagnostic Cable to work correctly.
Download Windows Drivers, unzip and run the setup to install.

If you have problems installing the driver or identifying the COM port used by the Diagnostic Cable read the Installation Guide
